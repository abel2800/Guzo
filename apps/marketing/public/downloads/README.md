# GUZO mobile builds

Place preview APK / IPA files here after running EAS builds:

| File | App |
|------|-----|
| `guzo-customer-android.apk` | Customer Android |
| `guzo-driver-android.apk` | Driver Android |
| `guzo-merchant-android.apk` | Merchant Android |
| `guzo-customer-ios.ipa` | Customer iOS (ad-hoc / internal) |

Then set `available: true` in `src/data/app-downloads.ts` or run `npm run copy:mobile-builds`.
